# 陈子龙找屎粑粑

一个平面 2D 平台跳跃网页小游戏，风格类似马里奥。

## 怎么玩

1. 用浏览器打开 `index.html` 文件。
2. 点击“开始游戏”。
3. 控制陈子龙在复杂地图中收集所有 💩 屎粑粑即可通关。

## 操作

- **A / D** 或 **← / →**：左右移动
- **W / ↑ / 空格**：跳跃
- **R**：自杀重来（扣一条命）

## 游戏要素

- ❤️ **3 条命**：血条耗尽会扣一条命，生命归零则游戏结束。
- 💖 **血包**：拾取后恢复 40 点血量。
- 🟣 **地面史莱姆敌人**：来回巡逻，碰到会受伤。
- 🦇 **飞行蝙蝠敌人**：沿路线飞行，碰到会受伤。
- ⚙️ **移动电锯**：横向移动，伤害很高。
- 🔺 **尖刺陷阱**：踩到会扣血。
- 🗺️ **小地图**：右上角可以看到自己在地图中的位置。

## 文件说明

- `index.html`：完整的游戏文件，包含 HTML、CSS 和 JavaScript，直接用浏览器打开即可游玩。

## 技术

- HTML5 Canvas
- 原生 JavaScript
- 无外部依赖
